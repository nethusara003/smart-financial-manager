Abstract      
 
Personal financial management remains challenging for individuals who must track expenditure, manage budgets, service debt, and plan for retirement simultaneously. Existing tools typically address these needs in isolation; few combine statistical expense forecasting, probabilistic retirement simulation, and AI-assisted contextual guidance within a single platform. 
 
This study aimed to design, develop, and evaluate the Smart Financial Tracker, a full-stack web application integrating multi-technique expense forecasting, a machine learning-driven retirement planning engine, and a large language model-based financial awareness assistant. 
 
Expense forecasting was implemented using linear regression for trend detection and a weighted moving average hybrid model, validated through rolling window backtesting that computed Mean Absolute Error, Root Mean Squared Error, and Mean Absolute Percentage Error per forecast window. The retirement planning module combined deterministic projection, Monte Carlo simulation for probability estimation, and a Random Forest Regressor trained on historical transaction data to predict income and expenses over a twelve-month planning horizon. The AI assistant aggregated each user’s complete financial profile at query time and queried a Groq-hosted large language model to deliver informational financial awareness guidance; the system does not provide regulated financial advice. 
 
System evaluation was conducted through backend unit and integration tests using Jest, frontend component tests using Vitest, and end-to-end workflow tests using Playwright, with all major modules passing functional testing. Rolling window backtesting confirmed that forecast confidence scoring operated as designed across tested transaction datasets; however, aggregate error figures against a formal held-out benchmark were not reported within the scope of this study. These findings indicate that integrating expense forecasting, probabilistic retirement planning, and LLM-assisted financial guidance within a unified platform is technically feasible and functionally verifiable; broader evaluation with representative user populations would be required before claims regarding effectiveness in supporting financial decision-making can be made
